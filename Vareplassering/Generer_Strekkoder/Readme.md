<h3>Beskrivelse</h3>
Denne genererer strekkoder sammen med strekkode verdien for bruk ved identifisering av vareplassering.
Strekkodene skrives ut på A4 ark, klippes opp og festes på hyller.

<h3>Dependencies:</h3>
  python-barcode
  Pillow
