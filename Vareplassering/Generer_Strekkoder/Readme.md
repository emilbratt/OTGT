<h3>Beskrivelse</h3>
<p>Denne genererer strekkoder sammen med strekkode verdien for bruk ved identifisering av vareplassering.</p>
<p>Strekkodene skrives ut på A4 ark, klippes opp og festes på hyller.</p>

<h3>Dependencies:</h3>
<p>python-barcode</p>
<p>Pillow</p>
